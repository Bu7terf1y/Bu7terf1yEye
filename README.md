# Bu7terf1yEye - 雪山盟资产探测工具

一款功能强大的资产探测工具，支持子域名枚举、端口扫描、存活检测和ICP查询等功能，采用多线程技术提高探测效率。

## 功能特性

- **子域名枚举**：
  - 支持多线程子域名枚举
  - 实时显示和写入结果
  - 可自定义线程数（默认10线程）
  - 结果保存在 `Subdomain` 目录

- **端口扫描**：
  - 支持多线程端口扫描
  - 可选择 Top100 或 Top1000 端口
  - 实时显示端口开放状态
  - 结果保存在 `Port` 目录的 CSV 文件中

- **存活检测**：
  - 支持多线程 URL 存活检测
  - 实时显示状态码、标题和响应长度
  - 支持输入探测和批量探测
  - 结果保存在 `Alive` 目录的 CSV 文件中

- **ICP查询**：
  - 查询域名的 ICP 备案信息
  - 显示备案号、主办单位、审核时间等
  - 结果保存在 `ICP` 目录
